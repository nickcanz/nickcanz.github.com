The Other Philadelphia

This project was built on 25 Feb for Code Across America Hackathon run by the folks at Code for America. The idea for the project was to make particular statistics about Philadelphia (unemployment, poverty etc) more real for people who aren't familiar with those stats. The site takes a random sampling of your Facebook friends and "tags" them with "Unemployed" or "High School Dropout" to make that data really hit home.

Colors used from Ethan Schoonover's awesome Solarized color theme.
